# Machine Learning Classification Projects

This repository contains a collection of machine learning classification projects built to explore how different models perform across a variety of datasets.

---

## Objective

The goal of this repository is to:

- Apply multiple classification models to the same dataset
- Understand how model performance changes with different data characteristics
- Emphasize the importance of data preprocessing and feature engineering
- Build a structured experimentation workflow for classification problems

---

## Experiment Approach

Each project in this repository follows a consistent workflow:

1. Data Understanding  
2. Data Cleaning & Preprocessing  
3. Feature Engineering (if needed)  
4. Model Training  
5. Model Evaluation & Comparison  

All implementation details, experiments, and results are documented within each project notebook.

---

## What You Will Find

- Multiple datasets from different domains  
- Consistent experimentation across projects  
- Performance comparison between models  
- Insights on how data impacts model behavior
